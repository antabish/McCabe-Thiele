# McCabe-Thiele Distillation Design

McCabe-Thiele graphical method for binary distillation analysis, based on **Example 7.1** from:

> Seader, J.D., Henley, E.J., & Roper, D.K.  
> *Separation Process Principles*, 3rd Edition

---

## System

**Benzene / Toluene** binary distillation at 1 atm, using experimental VLE data from Seader et al., Table 7.1.

---

## What the Code Does

Running `Example7.1.py` generates three publication-quality figures:

| Figure | Description |
|--------|-------------|
| **A** | Minimum stages at total reflux (Fenske condition) |
| **B** | Minimum reflux condition — pinch point construction |
| **C** | Actual operating design with equilibrium stage stepping |

All figures are exported as **PNG (600 dpi)** and **SVG** for use in presentations or reports.  
Stage coordinates are also exported to **`stage_steps.csv`** for import into PowerPoint or other tools.

## Background

The **McCabe-Thiele method** is a graphical technique for determining the number of theoretical stages required for a binary distillation. It combines:

- The **vapour-liquid equilibrium (VLE) curve**
- The **45° diagonal** (y = x line)
- The **rectifying** and **stripping operating lines**
- The **q-line** (feed condition line)

Stage stepping between the equilibrium curve and the operating lines gives the number of theoretical trays.
